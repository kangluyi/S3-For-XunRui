# 關於 S3 For XunRuiCMS

## 項目簡介

S3 For XunRuiCMS 是為迅睿CMS系統提供 AWS S3 雲存儲支持的插件。

## 功能特性

- 完整的文件上傳/下載管理功能
- 支持大文件分片上傳和斷點續傳
- 與迅睿CMS文件管理系統無縫集成
- 完善的錯誤處理和日誌記錄

## 安裝要求

- PHP 7.4 或更高版本
- 迅睿CMS 5.0+

## 安裝方式

通過迅睿CMS插件市場直接安裝

## 使用方法

1. 安裝插件後，在迅睿CMS後台配置 AWS S3 相關信息
2. 啟用插件後，即可在迅睿CMS文件管理中使用 AWS S3 作為存儲
