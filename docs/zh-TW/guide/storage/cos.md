# 騰訊雲COS存儲配置

- [官方網站](https://curl.qcloud.com/EUd4Xp2x)
- [定價說明](https://curl.qcloud.com/6zXLdsJd)

## 配置信息

1. 前往 [COS控制台](https://curl.qcloud.com/tPoI2o1J) 創建一個存儲桶
2. `Region` 填寫 所屬地域括號裡的英文部分
3. `Bucket` 填寫 存儲桶名稱
4. `Endpoint` 填寫 訪問域名 並去掉頭部的存儲桶名稱
5. 前往 [訪問密鑰 - 控制台](https://curl.qcloud.com/svYftakQ) 獲取密鑰，將`SecretId`填入`Access Key`，將`SecretKey`填入`Secret Key`
