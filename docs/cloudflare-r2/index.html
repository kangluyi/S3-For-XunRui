    配置信息#

    1. 首先去 [Cloudflare R2](https://link.guoctang.com/?target=https://www.cloudflare.com/developer-platform/products/r2/ "Cloudflare") 创建一个存储桶

    2. 填写 Access Key 和 Secret Key

    3. Region 填写 位置括号里的英文部分

    4. Bucket 填写 创建存储桶时填写的存储桶名称

    5. Endpoint 填写 S3 API并去掉结尾的存储桶名称

    6. 存储路径 填写文件的存储位置（默认为根目录）
