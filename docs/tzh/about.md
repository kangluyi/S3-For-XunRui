# S3 For XunRuiCMS相關

## 結構介紹

S3 For XunRuiCMS 是為迅睿CMS系統提供AWS S3雲存儲支援的插件。

## 運作特特性

- 完整的文件傳輸傳輸管理功能
- 支持大文件分片傳輸传传输和斷傳片續傳續
- 与迅睿CMS文件管理系統系統无缝集成
- 完善的錯誤處理錯誤处理和日誌記錄記錄

## 安裝安裝要求

- PHP 7.4 或更高版本

- 迅睿CMS  ##或++

## 安裝安裝方式
------

通過迅睿CMS插件市場直接安裝

## 使用方法

. 安裝插件後，迅睿CMS後台配置 AWS S3 相關資訊
.
. 啟用插件後，即可在迅睿CMS文件管理中使用 AWS S3 作為存儲
