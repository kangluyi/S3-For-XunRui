## Configuration

1. First, create a bucket on [Qiniu Cloud](https://portal.qiniu.com/kodo/bucket)
2. Enter the bucket name in `Bucket`
3. For `Endpoint` and `Region`, refer to [Service Domain](https://developer.qiniu.com/kodo/4088/s3-access-domainname)
4. Go to [Qiniu Cloud - Key Management](https://portal.qiniu.com/developer/user/key) to copy AccessKey (AK) and SecretKey (SK), and enter them in `Access Key` and `Secret Key`
