# Qiniu Cloud Kodo

- Official Website: [Qiniu Cloud](https://s.qiniu.com/A3UrAn)
- Pricing Information: [Price Document](https://www.qiniu.com/prices/kodo)

## Configuration Information

1. First, go to [Qiniu Cloud](https://portal.qiniu.com/kodo/bucket) to create a bucket.
2. Fill in the `Bucket` with the name of your bucket.
3. Fill in the `Endpoint` and `Region`, which can be referenced from [Service Domains](https://developer.qiniu.com/kodo/4088/s3-access-domainname)
4. Go to [Qiniu Cloud - Key Management](https://portal.qiniu.com/developer/user/key) to copy your Access Key (AK) and Secret Key (SK), and fill them into `Access Key` and `Secret Key`.

2. 将桶名称填入`Bucket`
3. `Endpoint`和`Region` ，可参考 [服务域名](https://developer.qiniu.com/kodo/4088/s3-access-domainname)
4. 前往 [七牛云 - 密钥管理](https://portal.qiniu.com/developer/user/key) 复制 AccessKey (AK) 和 SecretKey (SK) ，并填入`Acess Key`和`Secret Key`
