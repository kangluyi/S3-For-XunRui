## 配置信息

1. 登陆 [七牛云控制台](https://portal.qiniu.com/)
2. 填入`Bucket`参数（即步骤一创建的桶名）
3. 填入`Endpoint`和`Region`参数（参考 [服务域名](https://developer.qiniu.com/kend/4888/s3-access-domainname)）
4. 前往 [密钥管理](https://portal.qiniu.com/developer/user/key) 页面复制 AccessKey (AK) 和 SecretKey (SK) 并填入`AccessKey`和`SecretKey`

## Using K K
