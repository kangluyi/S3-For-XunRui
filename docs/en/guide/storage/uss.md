# Yuhu Cloud USS

## Configuration Information
1. Register on [Yuhu Cloud](https://console.upyun.com/register/?invite=qjG-JJP1q) and create a bucket.
2. Enter the bucket name in `Bucket`.
3. Fill in `Endpoint` with "https://s3.api.upyun.com".
4. Fill in `Region` with "us-east-1".
5. Under Storage Management — scroll down to Operator Authorization — S3 Credentials ![S3 Credentials](../../../zh-CN/guide/storage/uss/1.png). Enter the `AccessKey` and `SecretAccessKey` into `Access Key` and `Secret Key`.

2. 将桶名称填入`Bucket`
3. `Endpoint` 填入 “https://s3.api.upyun.com”
4. `Region` 填入 “us-east-1”
5. 存储管理 —— 下拉到操作员授权—— S3 凭证![S3 凭证](../../../zh-CN/guide/storage/uss/1.png)将`AccessKey`和`SecretAccessKey`填入`Access Key`和`Secret Key`
