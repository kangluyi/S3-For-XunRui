# YUPU Cloud USS

## Configuration Information
1. Register on [YUPU Cloud](https://console.upyun.com/register/?invite=qjG-JJP1q) and create a bucket.
2. Enter the bucket name in `Bucket`.
3. Enter "https://s3.api.upyun.com" in `Endpoint`.
4. Enter "us-east-1" in `Region`.
5. Go to Storage Management — scroll down to Operator Authorization — S3 Credentials ![S3 Credentials](./uss/1.png) and fill in the `AccessKey` and `SecretAccessKey` in `Access Key` and `Secret Key`.

2. 将桶名称填入`Bucket`
3. `Endpoint` 填入 “https://s3.api.upyun.com”
4. `Region` 填入 “us-east-1”
5. 存储管理 —— 下拉到操作员授权—— S3 凭证![S3 凭证](./uss/1.png)将`AccessKey`和`SecretAccessKey`填入`Access Key`和`Secret Key`
