# UCloud USS

## Configuration Information
1. Register [Register UCloud](https://console.upyun.com/register/?invite=qjG-JJP1q) and create a storage bucket.
2. Enter the bucket name into `Bucket`.
3. Enter `"https://s3.api.upyun.com"` into `Endpoint`.
4. Enter `"us-east-1"` into `Region`.
5. In Storage Management — scroll down to Operator Authorization — S3 Credentials ![S3 Credentials](./uss/1.png) Enter your `AccessKey` and `SecretAccessKey` into `Access Key` and `Secret Key`.
