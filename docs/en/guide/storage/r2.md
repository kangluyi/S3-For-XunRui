# Cloudflare R2

## Configuration Information

1. First, go to [Cloudflare R2](https://www.cloudflare.com/developer-platform/products/r2/) to create a storage bucket.

2. Fill in the `Access Key` and `Secret Key`![Cloudflare R2 Configuration](../../../zh-CN/guide/storage/r2/1.webp)

3. Set the `Region` to the English part inside the brackets.

4. Set the `Bucket` to the name of the bucket you created.

5. Set the `Endpoint` to the `S3 API` URL, removing the bucket name at the end.

6. Set the `Storage Path` to the location where files will be stored (defaults to the root directory).
