# Tencent Cloud COS Storage Configuration

- [Official Website](https://curl.qcloud.com/EUd4Xp2x)
- [Pricing](https://curl.qcloud.com/6zXLdsJd)

## Configuration

1. Go to [COS Console](https://curl.qcloud.com/tPoI2o1J) to create a bucket
2. Enter the region (English part in parentheses) in `Region`
3. Enter the bucket name in `Bucket`
4. Enter the access domain in `Endpoint` and remove the bucket name from the beginning
5. Go to [Access Keys - Console](https://curl.qcloud.com/svYftakQ) to get the keys, enter `SecretId` in `Access Key` and `SecretKey` in `Secret Key`
