## Configuration Information

1. Go to the [COS Console](https://curl.qcloud.com/tPoI2o1J) to create a bucket.

2. Fill in the `Region` with the English description in parentheses.
3. Fill in the `Bucket` with the bucket name.
4. Fill in the `Endpoint` with the access domain name, removing the bucket name at the beginning.
 ![Tencent Cloud COS](./cos/1.webp)

5. Go to the [Access Key - Console](https://curl.qcloud.com/svYftakQ) to get the keys, fill in the `SecretId` in `Access Key`, and fill in the `SecretKey` in `Secret Key`.
