# About S3 For XunRuiCMS

## Project Overview

S3 For XunRuiCMS is a plugin that provides AWS S3 cloud storage support for the XunRuiCMS system.

## Features

- Complete file upload/download management
- Support for large file chunked upload and resumable transfer
- Seamless integration with XunRuiCMS file management system
- Comprehensive error handling and logging

## Requirements

- PHP 7.4 or higher
- XunRuiCMS 5.0+

## Installation

Install directly from the XunRuiCMS plugin marketplace

## Usage

1. After installing the plugin, configure AWS S3 information in XunRuiCMS backend
2. After enabling the plugin, you can use AWS S3 as storage in XunRuiCMS file management
