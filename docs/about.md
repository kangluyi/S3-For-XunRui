# 关于 S3 For XunRuiCMS

## 项目简介

S3 For XunRuiCMS 是为迅睿CMS系统提供 AWS S3 云存储支持的插件。

## 功能特性

- 完整的文件上传/下载管理功能
- 支持大文件分片上传和断点续传
- 与迅睿CMS文件管理系统无缝集成
- 完善的错误处理和日志记录

## 安装要求

- PHP 7.4 或更高版本
- 迅睿CMS 5.0+

## 安装方式

通过迅睿CMS插件市场直接安装

## 使用方法

1. 安装插件后，在迅睿CMS后台配置 AWS S3 相关信息
2. 启用插件后，即可在迅睿CMS文件管理中使用 AWS S3 作为存储
