# 阿里雲 OSS

## 配置信息
1. 到阿里雲創建一個存儲桶

2. 將桶名稱填入`Bucket`
3. 在存儲桶概覽頁，找到「訪問端口」一欄，將「外網訪問 - Endpoint（地區節點）」複製到插件面板的`Endpoint`，可參考 [OSS 地區和訪問域名](https://help.aliyun.com/zh/oss/user-guide/regions-and-endpoints)
4. `Region` 參考 [OSS 地區和訪問域名](https://help.aliyun.com/zh/oss/user-guide/regions-and-endpoints) 中的「地區ID」進行填寫
5. `附件訪問URL` 填寫「外網訪問 - Bucket 域名」

6. 前往 [AccessKey](https://ram.console.aliyun.com/profile/access-keys) 复制 Access Key ID 和 Access Key Secret ，並分別填入`Access Key`和`Secret Key`
