# 腾訊雲 COS

- [官方網站](https://curl.qcloud.com/EUd4Xp2x)
- [定價說明](https://curl.qcloud.com/6zXLdsJd)

## 配置資訊

1. 前往 [COS控制台](https://curl.qcloud.com/tPoI2o1J) 創建一個存儲桶

2. `Region` 填寫 所屬地域括號里的英 văn部分
3. `Bucket` 填寫 存儲桶名稱
4. `Endpoint` 填寫 資 guit 應域名 幫去掉頭部的存儲桶名稱
 ![騰訊雲 COS](./cos/1.webp)

5. 前往 [訪問密鑰 - 控制台](https://curl.qcloud.com/svYftakQ) 获取密鑰，將`SecretId`填入`Access Key`，將`SecretKey`填入`Secret Key`
