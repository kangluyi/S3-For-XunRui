## 配置信息

1. � � 往 [COS控制台](https://curl.qcloud.com/tPoI2o1J) 創建一個存儲桶

2. `Region` 填寫 所屬地域括號里的英文部分
3. `Bucket` 填寫 存儲桶名称
4. `Endpoint` 填寫 變ien域名 并去掉頭部的存儲桶名称
 ![騰訊雲 COS](../../../zh-CN/guide/storage/cos/1.webp)

5. � � 往 [訪問密鑰 - 控制台](https://curl.qcloud.com/svYftakQ) 获取密鑰，將`SecretId`填入`Access Key`，將`SecretKey`填入`Secret Key`
