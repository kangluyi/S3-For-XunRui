# S3 For XunRuiCMS

## 將項目標複

S3 For XunRuiCMS 是一個支援迅睿CMS系統使用 AWS S3 云存储的插件。

## 程式特性

- 完整的文件上傳/下載管理功能
- 支援大文件分片上傳和斷點續傳
- 无缝集成到迅睿CMS文件管理系统
- 完善的錯誤處理和日誌記錄

## 設置需求

- PHP 7.4 或更高版本

- 迅睿CMS 5.0+

## 安裝方式

透過迅睿CMS插件市場直接安裝

## 使用方法

1. 安装插件后，在迅睿CMS后台配置 AWS S3 相关信息
2. 启用插件后，即可在迅睿CMS文件管理中使用 AWS S3 作为存储
