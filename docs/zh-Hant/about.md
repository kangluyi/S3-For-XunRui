# S3 For XunRuiCMS

## 項目簡介

S3 For XunRuiCMS 是一個支援迅睿CMS系統使用 AWS S3 雲存儲的插件。

## 程式特性

- 完整的文件上傳/下載管理功能
- 支援大文件分片上傳和斷點續傳
- 無縫集成到迅睿CMS文件管理系統
- 完善的錯誤處理和日誌記錄

## 安裝需求

- PHP 7.4 或更高版本
- 迅睿CMS 5.0+

## 安裝方式

透過迅睿CMS插件市場直接安裝

## 使用方法

1. 安裝插件後，在迅睿CMS後台配置 AWS S3 相關信息
2. 啟用插件後，即可透過迅睿CMS文件管理使用 AWS S3 作為存儲
