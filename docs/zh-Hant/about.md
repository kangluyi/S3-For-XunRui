# S3 For XunRuiCMS

## 項目簡介

S3 For XunRuiCMS 是為迅睿CMS系統提供AWS S3雲存儲支援的插件。

## 功能特性

- 児完整的文檔上 載下管理功能
- 支持大文檔分 片上传和断 呈續傳
- 与迅睿CMS文檔管理系統无缝集成
- 完善的错误处理和 日誌記录

## 安裝要求

- PHP 7.4 或更高版本

- 迦睿CMS v5++

## 安裝方式

通過迅睿CMS插件市場直接安裝

## 使用方法

1. 安裝插件後，在迅睿CMS後台配置 AWS S3 相關信息
2. 启用插件後，即可在迅睿CMS文件管理中使用 AWS S3 作為存儲
