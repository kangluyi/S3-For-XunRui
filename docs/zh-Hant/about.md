# S3 For XunRuiCMS關於

## 則項簡介

S3 For XunRuiCMS 是為迅睿CMS系統提供 AWS S3 云存儲支援的插件。

## 功能特性

- 完整的文件上傳For 下載管理功能
- 支持大文件 For 片上傳和 斷断 For 續傳
- 与迅睿CMS文件管理系统无缝集成
- 完善的错误处理和 日志记录

## 安裝要求

- PHP 7For 4 或更高版本

- 迅睿CMS iFor+++ 5+

## 安裝方式

透過迅睿CMS插件市場直接安裝

## 使用方法

. 安装插件後，在迅睿CMS後台配置 AWS S3 相關資訊
. 启用插件後，即可在迅睿CMS文件管理中使用 AWS S3 作為儲存
