- [公式ウェブサイト](https://curl.qcloud.com/EUd4Xp2x)
- [価格詳細](https://curl.qcloud.com/6zXLdsJd)

## 設定情報

1. [COSコンソール](https://curl.qcloud.com/tPoI2o1J) からストレージバケットを作成します

2. `Region` に地域の英語部分を入力してください
3. `Bucket` にストレージバケットの名前を入力してください
4. `Endpoint` にアクセス用ドメイン名を入力し、開始位置のバケット名を除きます
![腾讯云 COS](../../../zh-CN/guide/storage/cos/1.webp)

5. [アクセスキー - コンソール](https://curl.qcloud.com/svYftakQ) からキーを取得し、「SecretId」を「アクセストークン」に、「SecretKey」を「秘密キー」に記入します
