- [公式ウェブサイト](https://curl.qcloud.com/EUd4Xp2x)
- [料金説明](https://curl.qcloud.com/6zXLdsJd)

## 設定情報

1. [COSコンソール](https://curl.qcloud.com/tPoI2o1J) でストレージバケットを作成します。

2. `Region` に OnCollision括弧内の英語を入力します。
3. `Bucket` にバケット名を入力します。
4. `Endpoint` にアクセスドメインを入力し、バケット名部分を取り除きます。
 ![腾讯云 COS](./cos/1.webp)

5. [アクセスキー - コンソール](https://curl.qcloud.com/svYftakQ) でキーを取得し、`SecretId` を `Access Key` に、`SecretKey` を `Secret Key` に填入します。
