## 設定情報

1. [COSコンソール](https://curl.qcloud.com/tPoI2o1J) に移動してストレージ桶を生成します

2. `Region` には所在地域の英語部分を入力します
3. `Bucket` にはストレージ桶の名前を入力します
4. `Endpoint` には訪問用ドメイン名を入力し、ストレージ桶名前を削除します
 ![腾讯云 COS](./cos/1.webp)

5. [アクセスキー - コンソール](https://curl.qcloud.com/svYftakQ) からキーを取得し、`SecretId`を`アクセスキー`に、`SecretKey`を`秘密鍵`に入力します
