# テンセントクラウドCOSストレージ設定

- [公式サイト](https://curl.qcloud.com/EUd4Xp2x)
- [料金説明](https://curl.qcloud.com/6zXLdsJd)

## 設定情報

1. [COSコンソール](https://curl.qcloud.com/tPoI2o1J)に移動してストレージポータルを作成します
2. `Region`には所属地域の括弧内の英語部分を入力します
3. `Bucket`にはストレージポータル名を入力します
4. `Endpoint`にはアクセスドメインを入力し、先頭のストレージポータル名を除きます
5. [アクセス鍵 - コンソール](https://curl.qcloud.com/svYftakQ)に移動して鍵を取得し、`SecretId`を`Access Key`に、`SecretKey`を`Secret Key`に入力します
