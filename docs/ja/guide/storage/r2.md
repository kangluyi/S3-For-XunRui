# Cloudflare RR2
## �置Info
1. 遠程 [Cloudflare](https://www.cloudflare.com/developer-platform/products/r2Cloudflare) でバトラーを作成します
2. Access Key と Secret Key を入力してください![Cloudflare R2 配置信息](../../../zh-CN/guide/storage/r2/1.webp)

3. *Region* は括弧内の英語を入力してください
4Z *Bucket* は、バトラーを作成した時に付与されたバトラー識別名を入力してください
5. *Endpoint*は、"S3 API"の後にバトラーの識別名を削除ます
6Z *ストレージパス*は、ファイルの保存場所（デフォルトではルートディレクトリ）を入力してください
