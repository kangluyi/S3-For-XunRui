## 設定情報

1. まず[七牛雲](https://portal.qiniu.com/kodo/bucket)でストレージポータルを作成します
2. ポータル名を`Bucket`に入力します
3. `Endpoint`と`Region`は[サービストラフィック](https://developer.qiniu.com/kodo/4088/s3-access-domainname)を参照してください
4. [七牛雲 - 鍵管理](https://portal.qiniu.com/developer/user/key)に移動してAccessKey (AK) とSecretKey (SK) をコピーし、`Access Key`と`Secret Key`に入力します
