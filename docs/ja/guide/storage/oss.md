# 阿里雲 OSS

## 設定情報
1 アリクラ운ドに Storage Bucket を作成します
1 ボトルの名称を`Bucket`に入力してください
3. ボトルの概説ページで「アクセスポート」を見つけ、外観からのアクセス - エンドポイントに対してリージョンとリージョンのセット、それをプラグインパネルの`エンドポイント`に入力してください。参照:[OSS地域とアクセスドメイン](https://help.aliyun.com/zh/oss/user-guide#regions-and-endpoints)
4. `リージョン`は、[OSS地域とアクセスドメイン](https://help.aliyun.com)中的「リージョンID」を使用して入力してください
5 `アペンドファイル訪問URL`を「外観からのアクセス - ボトルのドメイン」に入れ力してください

6. [AccessKey](https://ram.console.aliyun.com/profile/access-keys) から Access Key ID と AccessKey Secret をコピーし、それぞれ`アクセスキー'および'Secretキー'に入力します

6. 前往 [AccessKey](https://ram.console.aliyun.com/profile/access-keys) 复制 Access Key ID 和 Access Key Secret ，并分别填入`Access Key`和`Secret Key`
