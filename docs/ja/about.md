## 项目简介

S3 For XunRuiCMS 是为迅睿CMS系统提供 AWS S3 云存储支持的插件。

## 功能特性

- 完整的文件上传For 下载管理功能
- 支持大文件 分片上传和断点续传
- 与迅睿CMS文件管理系统无缝集成
- 完备的错误处理和日志记录

## 安装要求

- PHP 7.4 或更高版本

通過迅睿CMSプラグイン市場で直接インストール
