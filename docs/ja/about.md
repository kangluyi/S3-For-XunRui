# S3 For XunRuiCMS について

## プロジェクト概要

S3 For XunRuiCMS は AWS S3 によるクラウドストレージを XunRuiCMS プロジェクトに提供するプラグインです。

## 機能

- 一貫したファイルアップロード/ダウンロード管理機能
- 大容量ファイルの分割アップロードと中断からの再開支援
- XunRuiCMS ファイル管理システムとシームレスな統合
- 完璧なエラーハンドリングとログ記録

## インストール要件

- PHP 7.4 以上

- XunRuiCMS 5.0 以上

## インストール方法
------

迅睿CMSプラグイン市場からインストールしてください

## 使用手順

1. プラグインをインストールし、迅睿CMSのバックエンドでAWS S3に関する情報を設定します
2. プラグインを有効にすると、迅睿CMSのファイル管理でAWS S3を使用してストレージを管理できます
